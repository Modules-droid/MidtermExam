Skinner's Theory

As you can see in the images above the app features have a reward which allows the user to get a reward if he or she completes the question on each level and also by gaining more points the user will be able to unlock new reviewer with this feature skinner’s operant conditioning has been applied because operant conditioning is a method of learning that occurs through rewards and punishments.

Pavlov's Theory

The image above shows that the application has a store feature that serves as the food in classical conditioning, as a response students are motivated to gain enough points to unlock reviewers.

Reinforcement

Completing a task gives positive reinforcement, which gives more motivation to the student to complete more tasks. The images show whenever you get a mistake, you lose a point which means less points, less probability to unlock reviewers. The decrease of gained points will help the student to avoid wrong answers in the future. But, you are still allowed to retry answering the quiz.

Punishment

As you can in the image above the app feature which allows the user to realize or learn what should not be than because the app reviewer will not be unlock unless the user realize he/she dont have enough points to unlock it and also if the user committed mistakes on answering each of the questions there will be a deduction of points which is a punishment if the user got mistakes on each question.
